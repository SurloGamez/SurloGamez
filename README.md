### 🚀 Featured projects

 
**[2D Drifting Physics Sim](https://github.com/SurloGamez/Drifting-Simulation-V1.0.0)** — *Scratch*
Top-down driving sim with a physically accurate two-wheel model: effective mass from rigid-body constraints, a Pacejka-inspired tire force curve, and tunable front/rear differentials to get sustained, controllable drifts. *(Add a GIF — drifting footage sells this instantly.)*

**[Unity Pose Estimation](https://github.com/SurloGamez/Unity-pose-estimate)** — *C# / Unity / MediaPipe*
A Unity package that translate Mediapipe's pose estimate output into pose information for 3D rigs. Sequences of poses are stored in custom animation system that can be run independly from in-engine animation system. Bridges an ML pipeline to Unity's render loop.

**[Wacow — Fitness Tracking App](https://github.com/Oasis-NEU/s26-group-16)** — *React Native (Expo) / Supabase*
Cross-platform fitness app for logging workouts and tracking progress, with gamification (XP, streaks, achievements) to build consistency. Email/password auth with persistent sessions, a 3-step onboarding flow, a searchable library of 55+ exercises across 12 muscle groups, and live workout sessions that save sets/reps/weight to a Postgres backend with row-level security. Group project — built on a 6-table Supabase schema.
